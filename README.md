# pronotheme
Pronotheme is a plugin for firefox that allow to use a dark mode in Pronote website

![Pronotheme screenshot](/screenshot/screenshot.jpg "Pronotheme picture")

## How to use it ?

the plugin is downloadable at https://addons.mozilla.org/fr/firefox/addon/pronotheme. You can try it even without a Pronote account, just follow this link https://demo.index-education.net/pronote/eleve.html and click in Pronotheme icon on the top right of the screen.

## Edit this plugin

- Clone this repo

      git clone https://github.com/mathias4833/pronotheme.git
      
- Open a new tab and search [about:debugging#/runtime/this-firefox](about:debugging#/runtime/this-firefox)

- Select to load a module and choose any file in the folder
